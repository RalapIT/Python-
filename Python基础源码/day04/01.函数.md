# 函数
#### 实现一些功能的代码集合
```
语法:  def 函数名 (参数) :
         执行的代码
```

#### 程序从上到下执行 函数必须要调用才会执行

#### 函数的生命周期:调用的时候函数调用完毕,执行完毕后函数销毁

#### [我对于return和print的区别理解]()
```
   在函数中 return作为接收函数执行完成后的返回值,在以后可以调用其返回值进行操作
   而print仅仅将结果向控制台打印,其以后不能进行操作,他的值为none
   ```
#### 全局变量和局部变量
1. 函数中的变量一般都是局部变量
2. 函数中于局部变量的时候,就不会使用到全局变量
3. 如果非要使用全局变量  则就要使用[global]()
