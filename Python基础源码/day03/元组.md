# 元组
1. 和列表差不多,创建后不可修改
2. 有序且可重复
```
          tu  =  (1,2,3,4,5)
	 1. 元组依然可以切片
	        print(tu[3])
		print(tu[-3])
		print(tu[-2:-7:-2])
		#注意如果一个元组中只有一个元素 需要在元素的后面添加一个逗号

		2.元组常用函数
		 len // max // min
		 len(tu)>>>>元组的长度
		>增删改在这里都是非法操作,但是可以删除整个元组
		 tu.index("元素") 查找这个元素的索引
		 tu = (1,2,3,4,5,6,7,8)
		 3.元组的遍历
		for i in range(len(tu)-1,-1,-1):
				print(tu[i])
		for i in range(-1,-len(tu)-1,-1):
				print(tu[i])
```
